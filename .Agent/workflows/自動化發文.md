# 自動化發文

這個工作流程會幫你自動爬取職場/轉職相關新聞，並將其改寫成適用於你網站的 SEO 文章，同時生成適合職場情境的封面圖，最後發佈到你的 Weebly。

1. **第一階段：爬取轉職新聞**  
   使用 `@1_爬取轉職新聞` 篩選數位時代、CakeResume、Inside 等來源。

2. **第二階段：改寫成 SEO 文章**  
   使用 `@2_撰寫轉職SEO文章` 把第一階段的新聞改寫為 HTML 格式的轉職類文章，包含 SEO 標題與描述，結尾並帶有延伸閱讀連結。

3. **第三階段：生成封面圖**  
   使用 `@3_生成轉職封面圖` 生成 2 張適合面試、辦公室、自我成長情境的封面圖片。

4. **第四階段：自動發佈到 Weebly**  
   使用 `@4_發佈到Weebly` 呼叫你的腳本來發布這篇完成的 HTML 內容與圖片。
